# Bedtime Story

A tiny app that picks a random bedtime story for your child — so you never have to answer "which one tonight?" again.

## What it does

You open the app, choose a mood (Animals, Magic, Space, Friendship, Adventure, or Dreams), hit the button, and get a short story ready to read aloud. Already read it? The app remembers and won't pick it again until you've gone through them all. You can also browse all the stories at once in the gallery.

## Running locally

```bash
npm install
npm run dev
```

That's it. Opens at `http://localhost:5173`.

## Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Vite
- Framer Motion for animations
- canvas-confetti (because picking a story should feel like a little celebration)
